# Restaurant Management System (C)

A console-based system designed to manage tables, menu items, customer orders, and billing in a restaurant environment.  
Developed as part of an academic project to practice data structures and modular programming in C.

---

## Features
- Manage restaurant tables (available/occupied)
- Add, view, and update menu items
- Track customer orders using queues and linked lists
- Generate bills and release tables
- Organized and modular code structure following SET Coding Standards

---

## Technologies
- Language: C  
- IDE: Visual Studio  
- Concepts: Hash Table, Queue, Linked List, Circular List  
- Platform: Windows Console Application

---

## How to Run
Open the project in Visual Studio, build it, and press Run to start the program.  
Alternatively, double-click the compiled 'RestaurantApp.exe' file to launch the console application.
